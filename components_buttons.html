Buttons

Buttons are a user interface component that help a user perform an action.

Usage

Buttons should be used to reinforce user action and need to be placed wherever a user is asked to perform an action, such as 'Contact' or 'Learn More'


Accessibility

Buttons should always utilize the appropriate HTML element, and should never be coded as links or nav items.


Examples

Primary Button

Primary buttons should be used for primary user actions i.e. Contact or first level information architecture.



<button type="button" class="btn btn-primary"> BUTTON TEXT </button>

Secondary Button

Secondary Buttons can be used on interior pages or as a second option for users along side a primary action.



<button type="button" class="btn btn-secondary"> BUTTON TEXT </button>

Tertiary button

Tertiary Buttons should be used sparingly and typically only as an 'on page' action, such as loading more content.



<button type="button" class="btn btn-outline-secondary"> BUTTON TEXT </button>
