Dimensions & Grids

Bootstrap default container width is 1140px, which is a bit narrow.
Dimensions for desktop will require adjustments to the container property:

                  
                    .container {
                    max-width: 1640px;
                    }

  • Tablet can utilize the ‘.col-lg’ class to accomodate max width of 750px
  • Mobile can utilize the ‘.col-sm’ class to accomodate max width of 330px
  • Grid pattern should follow Bootstraps inherent twelve column layout with all default breakpoints
